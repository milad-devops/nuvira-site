# Nuvira Dark — WordPress Theme (DevOps / Dark / Glass)

A dark, tech-oriented, DevOps-themed WordPress starter theme scaffold.

Features
- Dark neon gradients and glassmorphic UI
- Timeline résumé template
- Skills with animated progress bars
- 3D hover cards and lightweight micro-interactions
- Lucide icons integrated (via CDN)
- SVG network line animations
- Templates: Front Page, Resume, Portfolio, Contact, Blog (index.php), Single
- Fully responsive
- Placeholder SVG images and avatar included
- Simple JS for mobile menu and animations

Installation
1. Copy the folder `nuvira-dark` to `wp-content/themes/`.
2. Activate the theme in WordPress admin.
3. Assign the Primary Menu via Appearance → Menus and create pages using the provided templates.
4. Replace placeholder images at `assets/images/` with your own assets.

Notes
- Lucide icon library is loaded via CDN. If you prefer local hosting, download and enqueue locally.
- Contact template contains a basic HTML form. For production use, integrate with a plugin or server-side handler (e.g., Contact Form 7, WPForms).
